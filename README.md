# 图书版权页识别系统

这是一个基于Python开发的图书版权页识别桌面应用程序，可以自动识别并提取图书版权页中的关键信息。

## 功能特点

- 支持上传图书版权页图片
- 自动识别版权页文字信息
- 智能提取关键信息（ISBN、作者、出版社等）
- 清晰展示识别结果
- 用户友好的图形界面

## 安装说明

1. 确保已安装Python 3.9或更高版本
2. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行程序：
   ```bash
   python main.py
   ```
2. 点击"上传图片"按钮选择版权页图片
3. 点击"识别版权页"按钮开始识别
4. 查看右侧面板显示的识别结果

## 识别字段

- ISBN
- 书名
- 作者
- 出版社
- 出版时间
- 版次
- 印次
- 定价

## 技术栈

- PyQt5：构建桌面GUI界面
- OpenCV：图像处理
- PaddleOCR：文字识别
- Python正则表达式：信息提取
